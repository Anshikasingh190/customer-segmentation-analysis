# Customer Segmentation Analysis

##  Overview

This project focuses on analyzing customer data to identify different customer segments based on purchasing behavior. The goal is to help businesses improve targeting, marketing strategies, and customer retention.

##  Objective

* Segment customers into distinct groups using clustering techniques
* Identify high-value customers
* Provide actionable business insights

## Tools & Technologies

- Python (Pandas, NumPy, Matplotlib, Seaborn, Plotly)
- Machine Learning (K-Means Clustering, Scikit-learn)
- Streamlit (interactive web app)
- SQLite (user authentication)

## How to Run

1. Clone the repository
   git clone https://github.com/Anshikasingh190/customer-segmentation-analysis.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run customerscope_app.py

4. Open browser at http://localhost:8501

##  Key Steps

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection
* K-Means clustering implementation
* Visualization of customer segments

##  Results

* Identified multiple customer segments based on behavior
* Highlighted high-value and low-value customer groups
* Provided insights for targeted marketing strategies

##  Business Impact

* Helps improve customer retention
* Enables personalized marketing
* Supports data-driven decision-making
